## 🏁 프로젝트 시작 과정 
1. 가상환경 생성 및 실행
```python
python -m venv <가상환경 폴더이름> # 사용 할 수 있는 가상 환경 생성
. <가상환경 폴더>/scripts/activate # 가상 환경 실행 - Mac의 경우 scripts 대신 bin 폴더
```
2. Django Project 시작
```python
django-admin startproject projectname . 
# 이 방식은 협업에 좋은 구조화 방식을 만듬. 불필요한 폴더 생성X 
```
